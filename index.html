English Material

System Information

In an information system, there are components that are interrelated and can later be controlled, analyzed, created visualizations, and convey information to an organization's intended system.

An information system is an integrated set of components for collecting, storing, and processing data and for providing information, knowledge, and digital products. In information systems there is learning about databases.

Database

What is a Database? A database is a collection of data that is organized and connected so that it can be accessed easily. Databases generally reside on computer devices and are managed automatically by a Database Management System (DBMS). In simple terms, a database is a collection of data or information that is stored systematically. Databases have an important role in devices for collecting information, data or files in an integrated manner.

Database Function

1. Avoid data duplication :

The function of the database is to avoid duplicate data being stored. The DBMS can be set to recognize duplicate data when it is input and then provide notification to the database manager.

2. Centralized data :

Without a database, data would be scattered across various stores. With a database, we can collect all data in one place so monitoring is easier.

3. Save costs :

Because the data is centralized in one place, we don't need another storage location. Automatically, this will make us more cost effective.

4. Can be accessed by many users at once :

Databases store all data in one system so we no longer need to be confused about how to share access. Everyone we allow can access the database from practically anywhere.

5. Makes data identification easier :

From the large amount of data in the database, you can identify it easily. This is done by creating data grouping or filtering. So, when there is a request for data access, the information can be provided quickly according to the previously created categories.

6. Improve data security :

If we store important data in many safes, we will definitely need many keys. So, because this database is centralized, we only focus on one key but this can be maximized with layered protection. This will certainly make the data safer.
